Brush

Brush gets the current center coordinates from the array of centers and uses the brush radius to calculate which pixels to set to the selected color. Setting the pixel color is constratined by the distance between the pixel being set and the original center of the brush. If the pixel being changed is further from the center than the allocated radius, it is not changed because it would not be in the brush's area.
You can include as many results as you want, but please at least include the following results:
Push_Image=flower.jpg&Brush=16;[0.0968858131488,0.0968858131488,0.705882352941,1];x222y165x303y160x266y234

Push_Image=town.jpg&Brush=10;[1,1,1,1];x475y83x304y100x135y187

Soft Brush

Soft brush is similar to the implemented brush function. However, the color the image pixel is changed to is calculated using alpha blending formulas, which muliplies the opacity value with the current brush color added to the muliplied image pixel color and 1 minus the opacity. The opacity of the brush is linearly decreasing as the pixels get further from the center of the brush stroke.
You can include as many results as you want, but please at least include the following results:
Push_Image=flower.jpg&Soft_Brush=91;[1,1,1,1];1;x269y193
